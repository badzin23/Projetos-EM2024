HTML Forms

Formulário em HTML

Formulários HTML são um dos principais pontos de interação entre um usuário e um web site ou aplicativo. Eles permitem que os usuários enviem dados para o web site.

O que é um container?

Container é um elemento que irá englobar toda sua página, será o primeiro elemento e dentro dele virão as informações

Como funciona o container form?

O Form é um container que permite agrupar controles para entrada de dados

Como funciona o post?

O método POST cria um par nome/valor que são passados no corpo da mensagem de pedido HTTP.

Como funciona o INPUT?

Tag input cria diferentes campos de entrada de dados no documento HTML. No geral, input deve ser usada dentro da tag form, para que possa passar os dados coletados para algum outro documento através dos processos nativos de form.
